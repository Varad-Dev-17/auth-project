# 🔐 Auth Project

&gt; 🚀 Learning by building, breaking, fixing, and improving.

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- JWT (JSON Web Tokens)
- REST APIs
